# Schema for modules.tsv

Modules are described in `modules.descriptive.md`.


## Fields

- name (1st column) — Module name per data table instructions.
- operations — Operations that the module provides capacity for.


## Notes

1. A module's capacity can be shifted among its enabled operations. The speed and cost of this conversion are fixed properties of each module and can represent anything from a major refitting to a minor operational adjustment.
